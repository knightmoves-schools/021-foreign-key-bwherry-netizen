# 021 Foreign Key

[![021 Foreign Key](https://img.youtube.com/vi/ehijzv6m7Oo/0.jpg)](https://www.youtube.com/watch?v=ehijzv6m7Oo)

The SQL in the `exercise.sql` file
- Should create a table named `Stores` with the following columns `ID`, `LOCATION`, `EMPLOYEE_ID` and `REVENUE`
- Should designate any one of the columns as the primary key
- Should establish a foreign key that references the location of each store to the `Employee` table
